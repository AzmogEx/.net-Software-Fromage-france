# Gestion des Fromages de France

Ce projet est un logiciel développé en .NET permettant de répertorier et d'afficher les fromages de France, classés par région et par type. Ce logiciel est destiné aux amateurs de fromages et aux professionnels souhaitant explorer la richesse fromagère de la France.

## Fonctionnalités

* Affichage des fromages par région
* Filtrage par type de fromage (pâte molle, pâte dure, etc.)
* Recherche rapide par nom ou région
* Interface conviviale et intuitive

## Technologies utilisées

* .NET
* WPF (Windows Presentation Foundation) pour l'interface utilisateur
* Base de données locale pour le stockage des informations

## Installation

1. Cloner le dépôt :

   ```bash
   git clone https://github.com/AzmogEx/.net-Software-Fromage-france.git
   ```
2. Ouvrir le projet avec Visual Studio
3. Restaurer les packages NuGet
4. Lancer l'application depuis Visual Studio

## Utilisation

1. Sélectionner une région pour afficher les fromages associés
2. Utiliser la barre de recherche pour trouver un fromage spécifique
3. Filtrer par type de fromage pour affiner la recherche
